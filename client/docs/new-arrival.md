# New arrival component

#### Description
The new arrival component is responsible displaying and highlighting the latest 
arrivals or products in the application. It provides an engaging and visually 
appealing section to showcase the newest additions to the inventory. This 
document will guide you through the process of creating a new arrival component
effectively.

#### Usage
to use the new arrival component you will need to import it into your project
